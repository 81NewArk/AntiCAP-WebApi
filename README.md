## 🌍环境说明
```
python 3.8+
```

## 📁 安装

###  1.Git克隆仓库 或 手动下载

```
git clone https://github.com/81NewArk/AntiCAP-WebApi

cd AntiCAP-WebApi
```



### 2.使用清华源下载项目所需依赖
```
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```


### 3.运行main.py
```
python main.py
```


# 🔧 Web主页
服务器部署需要打开6688端口

http://localhost:6688/
